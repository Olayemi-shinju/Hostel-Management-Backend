# Hostel Management Backend API

This is a backend API for a Hostel Management System built with Node.js, Express, MongoDB, JWT authentication, and SendGrid for email services.

---

## 🚀 Features

- User registration & login
- Email verification
- Password reset functionality
- JWT authentication
- Secure cookies (HttpOnly)
- Role-based access (user/admin)
- SendGrid email integration
- MongoDB database

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB + Mongoose
- JWT (Authentication)
- SendGrid (Email service)
- bcryptjs (Password hashing)
- dotenv
